# Введение

## Актуальность темы

В условиях современного высшего образования возрастает потребность в эффективных инструментах анализа и прогнозирования академической успеваемости студентов. Традиционные подходы к оценке успеваемости, основанные на периодическом контроле и ручной обработке данных, не позволяют своевременно выявлять студентов группы риска и принимать обоснованные решения по их поддержке.

С развитием информационных технологий появляется возможность автоматизации процессов анализа образовательных данных и построения прогнозных моделей на основе исторических показателей успеваемости. Это открывает перспективы для создания интеллектуальных систем поддержки принятия решений в образовательном процессе.

Целью настоящего проекта является разработка веб-приложения для прогнозирования успеваемости студентов с использованием современных веб-технологий. Проект реализован на платформе Node.js с фреймворком Express, базой данных PostgreSQL и контейнеризацией в Docker для обеспечения простоты развертывания и масштабируемости.

## Проблематика исследования

При разработке систем прогнозирования успеваемости возникают следующие основные проблемы:

1. **Неоднородность данных** - различные форматы хранения информации об оценках и результатах обучения
2. **Сложность алгоритмов** - необходимость учета множества факторов, влияющих на успеваемость
3. **Требования к производительности** - обработка больших объемов данных в реальном времени
4. **Безопасность информации** - защита персональных данных студентов и результатов обучения
5. **Пользовательский опыт** - создание интуитивно понятных интерфейсов для разных категорий пользователей

## Цель и задачи исследования

**Цель работы:** Создание веб-приложения для прогнозирования академической успеваемости студентов с использованием современных технологий разработки и методов анализа данных.

Для достижения поставленной цели необходимо решить следующие задачи:

1. **Проектирование архитектуры системы** - разработка структуры базы данных, API интерфейсов и клиентской части приложения
2. **Реализация алгоритмов прогнозирования** - создание математических моделей для анализа успеваемости на основе оценок по экзаменам и зачетам
3. **Разработка пользовательского интерфейса** - создание интуитивно понятных интерфейсов для преподавателей и студентов с использованием React.js
4. **Обеспечение безопасности** - реализация системы аутентификации, авторизации и защиты персональных данных
5. **Контейнеризация решения** - настройка Docker для упрощения развертывания и обеспечения консистентности среды выполнения
6. **Тестирование и оптимизация** - проверка работоспособности системы и оптимизация производительности

## Объект и предмет исследования

**Объект исследования:** Процесс прогнозирования академической успеваемости студентов в высших учебных заведениях на основе анализа оценок по экзаменам и результатам зачетов.

**Предмет исследования:** Методы и технологии разработки веб-приложения для анализа образовательных данных и построения прогнозов успеваемости с использованием React.js, Node.js, PostgreSQL и Docker.

## Технологический стек проекта

Разрабатываемое веб-приложение построено на основе современных технологий:

**Frontend:**
- **React.js 18.2.0** - библиотека для создания пользовательского интерфейса
- **React Router DOM 6.3.0** - маршрутизация в одностраничном приложении
- **Tailwind CSS** - utility-first CSS фреймворк для стилизации
- **Axios 1.4.0** - HTTP клиент для взаимодействия с API
- **Recharts 2.7.2** - библиотека для создания графиков и диаграмм
- **React Hook Form 7.44.3** - управление формами
- **Lucide React 0.263.1** - набор иконок

**Backend:**
- **Node.js 14+** - серверная платформа JavaScript
- **Express.js 4.18.2** - веб-фреймворк для Node.js
- **PostgreSQL 15** - реляционная база данных
- **JWT (jsonwebtoken 9.0.2)** - аутентификация и авторизация
- **bcryptjs 2.4.3** - хеширование паролей
- **pg 8.11.3** - драйвер PostgreSQL для Node.js

**DevOps:**
- **Docker** - контейнеризация приложения
- **Docker Compose** - оркестрация многоконтейнерного приложения
- **PgAdmin 4** - веб-интерфейс для управления PostgreSQL

## Функциональность разрабатываемого приложения

Разрабатываемое веб-приложение предоставляет следующие возможности:

**Для преподавателей:**
- Управление студентами (добавление, редактирование, удаление)
- Ввод и управление оценками по экзаменам и зачетам
- Расчет прогнозов успеваемости для студентов
- Просмотр статистики по группам
- Фильтрация и поиск данных
- Анализ трендов успеваемости

**Для студентов:**
- Просмотр собственных оценок и результатов
- Получение персонального прогноза успеваемости
- Отслеживание динамики академических показателей

**Система ролей:**
- Разграничение доступа между преподавателями и студентами
- Безопасная аутентификация с использованием JWT токенов
- Защита персональных данных студентов

## Алгоритм прогнозирования

В разрабатываемом приложении реализован алгоритм прогнозирования, основанный на статистическом анализе:

1. **Анализ оценок по экзаменам** - расчет средней оценки с нормализацией (2-5 баллов → 0-100%)
2. **Анализ результатов зачетов** - расчет процента успешно сданных зачетов
3. **Взвешенная формула прогноза** - 70% экзамены + 30% зачеты для расчета общего показателя успеваемости
4. **Сохранение истории прогнозов** для отслеживания динамики

## Практическая значимость

Разработанное веб-приложение представляет собой комплексное решение для автоматизации процессов анализа и прогнозирования академической успеваемости. Система интегрирует современные веб-технологии с методами статистического анализа данных, обеспечивая эффективное управление образовательным процессом.

**Области применения:**
- **Образовательные учреждения** - мониторинг успеваемости студентов и выявление групп риска
- **Преподавательский состав** - получение аналитических данных для принятия педагогических решений
- **Студенты** - доступ к персональной аналитике успеваемости и прогнозам
- **Администрация вузов** - формирование отчетности и планирование образовательной деятельности

**Ключевые преимущества решения:**
- Автоматизация процессов анализа успеваемости
- Интуитивно понятный пользовательский интерфейс
- Масштабируемость и возможность интеграции с существующими системами
- Обеспечение безопасности персональных данных
- Простота развертывания и сопровождения
